#  Banking System Simulation

A mini banking system that demonstrates real-world financial operations using Object-Oriented Programming (OOP) principles.

---

##  Overview

This project simulates a basic banking environment where users can perform essential banking operations such as account creation, transactions, and balance management. It focuses on backend logic, clean code structure, and database integration.

---

##  Features

- User Account Management  
- Create Account (Savings / Current)  
- Deposit Money  
- Withdraw Money  
- Perform Transactions  
- Balance Inquiry  
- Transaction History  
- User Authentication (Username & Password)

---

##  Technologies Used

- Core Java (OOP) – Version 21  
- MySQL – Version 8.0  
- JDBC (Java Database Connectivity)  
- Java Collections Framework
- Java Swing for GUI

---


##  Project Flow

1. User registers or logs in  
2. Account is created (Savings/Current)  
3. User performs transactions (Deposit/Withdraw)  
4. Data is stored and managed using MySQL  
5. Users can check balance and transaction history
